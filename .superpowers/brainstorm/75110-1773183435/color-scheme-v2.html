Heatmap Color Gradient

Which gradient for signal strength? Weak → Strong (left to right on legend)

A

Thermal Spectrum (NetSpot style)

Blue (weak) → cyan → green → yellow → red (strong). The industry standard for network heatmaps. Most nuanced — you can see subtle gradations easily.

B

Stoplight (WiFiman style)

Red (weak) → orange → yellow → green (strong). Instantly readable — red means bad, green means good. Less nuanced but very intuitive.

C

Plasma (Scientific)

Dark indigo (weak) → purple → red → orange → yellow (strong). Beautiful, high contrast on dark backgrounds. Less familiar but striking.
