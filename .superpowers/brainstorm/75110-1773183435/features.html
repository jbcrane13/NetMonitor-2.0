Phase 1 Feature Scope

Select everything you want in Phase 1. We'll cut nothing that's checked — and nothing else goes in.

✓

Floor plan import

Import PNG/JPEG/PDF as the survey canvas. Already partially implemented.

✓

Scale calibration

Draw a line between two known points, enter real distance (ft or m). Enables distance-based stats.

✓

Live RSSI + noise floor (CoreWLAN)

Real-time signal meter in sidebar. RSSI, noise floor, SNR, channel, band, link speed — all from CoreWLAN.

✓

Click-to-record measurement points

Click anywhere on the canvas to stamp a measurement at that location using current RSSI.

✓

IDW heatmap overlay (Signal / SNR)

Inverse-distance-weighted interpolation rendered as a color gradient on top of the floor plan. Signal strength and SNR overlays minimum.

Multiple visualization overlays

Latency, download speed, upload speed, channel overlap, AP coverage — switchable from toolbar. Requires active mode data for speed/latency.

Active scan mode (speed + latency per point)

When enabled, each recorded point runs a quick gateway ping and short speed test. Slower but richer data.

Nearby AP list (CoreWLAN scan)

CoreWLAN can scan and return all visible access points with RSSI, channel, band, BSSID. Shows a list of detected APs in the sidebar.

Per-AP heatmap filter

When you have multiple APs, show the heatmap for a specific BSSID only. Lets you compare coverage zones of AP1 vs AP2.

Project save / load

Save named survey projects to disk (JSON + image). Open previous projects. Project list shown via File menu or open sheet.

Export heatmap as PNG/PDF

Save the rendered heatmap (floor plan + overlay + legend) as a file. Useful for reports.

Undo last measurement point

Cmd+Z removes the most recently recorded point. Essential for fixing misclicks.

Hover tooltip on measurement points

Hover over a recorded dot to see all its data (RSSI, SNR, channel, timestamp, speed if active). Like NetSpot's popover.

Coverage threshold indicator

Set a minimum acceptable RSSI (e.g., -70 dBm). Areas below threshold shown with a hatched red overlay. NetSpot's "zones of poor coverage."

Channel / band visualization

Color each measurement dot by frequency band (blue = 5 GHz, orange = 2.4 GHz, green = 6 GHz). Quick visual scan of band distribution.
