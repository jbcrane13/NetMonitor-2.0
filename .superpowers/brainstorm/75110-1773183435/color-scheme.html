A — Thermal Spectrum

Purple/blue (weak) → red → orange → yellow → green → cyan (strong). Most data-rich and professional. What NetSpot and most tools use.

-90 dBm
-30 dBm

B — Stoplight (Red → Green)

Red (weak) → orange → yellow → green (strong). Instantly readable. Less nuanced but very intuitive. Similar to WiFiman.

-90 dBm
-30 dBm

C — Plasma (Dark → Warm)

Indigo/dark (weak) → purple → red → orange → yellow (strong). Scientific/heat-map style. Beautiful but less familiar to network tools users.

All options support adjustable opacity. The gradient applies to signal strength overlay — SNR, latency, and speed use the same gradient adapted to their scale.
